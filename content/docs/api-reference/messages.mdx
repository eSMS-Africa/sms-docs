## POST /api/messages/send

Send a single SMS message.

**Auth:** Required

### Request body

```json
{
  "to": "+256712345678",
  "text": "Your OTP is 482910.",
  "sender_id": "MyApp",
  "route": "AT_UG"
}
```

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `to` | string | ✅ | Recipient phone in E.164 format |
| `text` | string | ✅ | Message content |
| `sender_id` | string | - | Approved sender ID. Defaults to route default |
| `route` | string | - | Route code. Auto-detected if omitted |

### Response `200`

```json
{
  "id": "msg-a1b2c3d4",
  "status": "submitted",
  "segments": 1,
  "cost": 35.00,
  "cost_currency": "UGX",
  "route": "AT_UG",
  "balance_after": 1965.00
}
```

### Errors

| Code | Detail |
|------|--------|
| 400 | `invalid_phone` - phone cannot be parsed |
| 400 | `no_route` - no active route for this country |
| 422 | `insufficient_balance` - wallet balance too low |

## POST /api/messages/send-bulk

Send a bulk SMS campaign.

**Auth:** Required

### Request body

```json
{
  "contact_list_ids": [1, 2],
  "text": "Hi {{name}}, your message here.",
  "sender_id": "MyApp",
  "schedule_mode": "now",
  "scheduled_at": null,
  "drip_rate": null
}
```

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `contact_list_ids` | array | ✅ | List IDs to send to |
| `text` | string | ✅ | Message text with optional `{{variables}}` |
| `sender_id` | string | - | Approved sender ID |
| `route` | string | - | Force a specific route |
| `schedule_mode` | string | - | `now` (default), `scheduled`, `drip` |
| `scheduled_at` | string | If scheduled | ISO 8601 UTC datetime |
| `drip_rate` | integer | If drip | Messages per minute |

### Response `200`

```json
{
  "batch_id": "batch-uuid",
  "total_recipients": 250,
  "estimated_cost": 1.88,
  "status": "sending"
}
```

## GET /api/messages

List messages with pagination and optional status filter.

**Auth:** Required

### Query params

| Param | Default | Description |
|-------|---------|-------------|
| `page` | `0` | 0-indexed page |
| `limit` | `20` | 1–100 |
| `status` | - | Filter: `queued`, `submitted`, `sent`, `delivered`, `failed` |

### Response `200`

```json
{
  "messages": [
    {
      "id": "msg-a1b2c3d4",
      "phone": "+256712345678",
      "text": "Hello",
      "sender_id": "eSMSAfrica",
      "route": "AT_UG",
      "country": "UG",
      "segments": 1,
      "cost": 35.00,
      "currency": "UGX",
      "status": "delivered",
      "error_code": null,
      "retry_count": 0,
      "created_at": "2026-05-02T12:00:00Z",
      "delivered_at": "2026-05-02T12:00:05Z"
    }
  ],
  "total": 150,
  "page": 0,
  "limit": 20
}
```

## POST /api/messages/:message_id/retry

Retry a failed message.

**Auth:** Required

Only works on messages with `status: "failed"`. Returns `409` if the message is not in a retryable state.

### Response `200`

```json
{
  "id": "msg-a1b2c3d4",
  "status": "retrying",
  "retry_count": 1
}
```

## GET /api/messages/export

Export messages as a CSV download.

**Auth:** Required

### Query params

| Param | Description |
|-------|-------------|
| `status` | Optional - filter by status |

Returns `Content-Type: text/csv` with columns: `message_id`, `phone`, `sender_id`, `text`, `status`, `country`, `currency`, `cost`, `segments`, `created_at`, `delivered_at`.
